# SSL Automation

AgentDomain provisions SSL through Cloudflare for SaaS custom hostnames. Each
agent gets exactly one apex hostname such as `example.com`; `www.example.com`
is not created.

## How it works

1. Create a Cloudflare custom hostname for the apex domain.
2. Store the Cloudflare custom hostname ID, hostname status, SSL status,
   validation records, and validation errors.
3. Write any Cloudflare validation TXT records through Spaceship Basic DNS.
4. Reconcile status from Cloudflare until both hostname and SSL status are
   `active`.
5. Delete the custom hostname when the agent is revoked, removed, or expires
   without renewal.

## Admin repair

```http
POST /api/v1/admin/agents/:id/ssl-reprovision
```

The repair endpoint recreates the apex Cloudflare custom hostname and stores
the replacement status in `ssl_hostnames`.
