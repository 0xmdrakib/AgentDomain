# Welcome to AgentDomain

AgentDomain gives any AI agent a complete internet identity: a real domain,
Base Basename, managed DNS, SSL, email inbox, and an AgentID NFT paid for with
USDC on Base.

## What you get in one transaction

- A traditional **domain name** (`.com`, `.ai`, `.xyz`, etc.) registered for up to 10 years
- A **Basename** on Base L2 (`.base.eth`)
- An optional **ENS name** on Ethereum L1 (`.eth`)
- An **AgentID NFT** (ERC-721) representing the identity bundle
- A managed **DNS zone** (Cloudflare)
- An auto-provisioned **SSL certificate** (Let's Encrypt)
- An **email inbox** (`agent@yourdomain.com`)
- A funded **Renewal Vault** to ensure the identity never expires

All paid in **USDC on Base** via the **x402** protocol. No accounts, no KYC,
no human required.

## Why?

The agent economy is exploding. Hundreds of services on Base accept payment
via x402, agents are transacting trustlessly, and frameworks like ElizaOS,
AgentKit, and LangChain are launching daily. But until now, agents have had no
standard way to obtain a verifiable identity that works across HTTP, Web3,
and email.

AgentDomain fills that gap with a single, composable, agent-native API.

## How it works

```
1. Agent calls /agents/register
2. Server returns 402 Payment Required (x402 challenge)
3. SDK signs USDC EIP-3009 transferWithAuthorization
4. Agent retries with X-Payment header
5. Server provisions: ENS -> Spaceship -> Cloudflare -> Resend -> Basenames -> Mint -> DB
6. Returns the registration result
```
