# ElizaOS

AgentDomain can be exposed as an ElizaOS action provider so agents can register
or inspect identities from inside their runtime.

## Example intent

```text
Register the name market-maker.ai with email enabled and a Basename.
```

## Result

The agent receives a domain, onchain identity, inbox, and SSL provisioning status
that can be added to its character profile or memory.
