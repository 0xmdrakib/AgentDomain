# Coinbase AgentKit

AgentDomain is designed for Base-native agents and pairs naturally with Coinbase
AgentKit wallets.

## Flow

1. AgentKit wallet signs x402 payment authorization.
2. AgentDomain settles USDC on Base.
3. The agent receives a domain, Basename, email, SSL, and AgentID NFT.

This gives AgentKit agents a stable public internet identity and an onchain
ownership primitive.
